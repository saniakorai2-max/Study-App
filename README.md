# Forever

Forever is a calm, aesthetic, and highly practical study tracking web application designed for lifelong learners. Instead of using complex, stressful tools, Forever provides a peaceful environment where you can organize your tasks and track your learning progress over time.

Designed with a responsive mobile-first approach, the app is fully optimized to be saved and used directly on an iPhone home screen or any web browser.

---

## Features

- **Aesthetic Glassmorphic UI:** A modern interface featuring a calm background palette and translucent "frosted glass" containers.
- **Main Study Dashboard:** An overview of your daily progress alongside individual subject tracking cards (Math, Physics, and Information Technology).
- **Interactive Todo List:** A fully functional task manager built with native JavaScript. You can easily add tasks, check them off as you complete them, and remove them to keep your workspace clean.
- **Mobile Responsive:** Built to look and feel like a native mobile app when accessed on mobile devices.

---

## Technologies Used

- **HTML5:** For page structure and semantic layout.
- **CSS3:** For responsive design, CSS Grid, Flexbox, and the glassmorphic styling.
- **JavaScript (ES6):** For interactive components, including the dynamic DOM manipulation used in the task list.

## Roadmap

- **Dynamic Date & Time:** Fetch the user's actual local system date automatically using JavaScript.

- **Active Subject Timers:** Create a feature that lets users start an active stopwatch when they study a specific subject.

- **Dynamic Subject Creation:** Functional "+" button to let users create and name their own custom study categories.

- **Local Storage Integration:** Saving tasks and accumulated study hours so they do not disappear when you refresh the page.
